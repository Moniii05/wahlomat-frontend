# HTWahl-O-Mat
ist eine Wahlentscheidungshilfe für Studierende. Mit dem Wahl-O-Mat der HTW Berlin kannst du deine Positionen
zu Hochschul-Themen mit denen der Kandidat:innen vergleichen.

## Inhaltsverzeichnis
- Installation
- Technologien (FE)

## Installation
Frontend Setup

1. FE Repository clonen:
```bash
git clone https://gitlab.rz.htw-berlin.de/s0592471/htw.wahlomat.git
```

2. Navigieren zum FE Verzeichnis:
`cd wahlomat.frontend`
`cd wahlomat`

3. Installieren der Abhängigkeiten:
`npm install`

4. FE starten:
`ng serve`

## Technologien
- Angular
- Bootstrap
