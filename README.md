Angular-Frontend für den HTWahl-O-Mat.  
Enthält die Oberflächen für Wählende, Kandidat:innen und Admins.

## Tech
- Angular 19
- TypeScript
- Bootstrap + Bootstrap Icons
- Transloco (i18n)

## Voraussetzungen
- Node.js (LTS empfohlen)
- npm

## Installation
```bash
npm install
```md
